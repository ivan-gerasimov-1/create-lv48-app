Astro site for {{displayName}}.

## Vite Integration

Astro uses Vite for its development server, module graph, and build pipeline. This template uses `rolldown-vite@7.3.1` as a temporary migration bridge to exercise the Rolldown build path before Astro has native Vite 8 support. The Vite alias is local to this workspace and can be removed once Astro supports Vite 8 natively.
