# Development

## Requirements

- Node.js 24 or newer
- npm

## Tests

Run tests:

```bash
npm test
```

Watch mode:

```bash
npm run test:watch
```

Tests use Vitest.

## Local Commands

Run build and typecheck watch mode:

```bash
npm run dev
```

Build the CLI:

```bash
npm run build
```

Run the CLI in development mode:

```bash
npm run start:dev
```
